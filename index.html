How it works

Web Watcher fetches the target URL through a CORS proxy, extracts comparable content, and hashes it. Each subsequent fetch is compared against the previous hash — if it changes, you get an audio alert, browser notification, and flashing tab title.

Tips:

  • Use Text content mode to ignore HTML noise (ad scripts, session tokens).
  • A CSS selector lets you focus on one part of the page — e.g. .showtimes.
  • Keyword watch mode alerts you only when specific words appear or disappear.
  • Keep this tab open (pinned is fine). A background Web Worker keeps the timer accurate.

Limitations: Sites behind heavy bot protection (Cloudflare challenges, CAPTCHAs) may not be fetchable through free CORS proxies. If all proxies fail, try a different proxy from the Advanced options.
